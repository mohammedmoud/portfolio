Core Features

Everything You Need to Collaborate

Powered by WebSockets for true bidirectional, low-latency real-time communication.

⚡

Real-Time Sync

Every keystroke is instantly broadcast to all room users via Socket.IO — zero refresh needed.

🚪

Room Management

Create private rooms with unique UUIDs. Share the ID to invite teammates instantly.

👥

Role System

Three roles: Creator, Editor, Viewer. The Creator controls who can edit or just view.

🔐

Auth System

Username & password registration. Session is tied to the Socket.IO connection.

🟢

Presence Tracking

See who's online or offline in real-time. Disconnected users are marked offline automatically.

🔗

Share Room Link

One-click Room ID copy. Invite others as Editors or Viewers with a simple toggle.

Flow

How It Works

From sign-up to real-time collaboration in seconds.

1

Sign Up / Login

Enter a username and password — no email, instant access.

2

Create a Room

Click "Create Editor" — the server returns a unique UUID room ID. You become the Creator.

3

Share the Room ID

Click "Share Editor" to copy the ID and send it to your teammates.

4

Collaborate in Real-Time

Any user types → file_change fires → Socket.IO broadcasts to every room member → all editors see it instantly.

5

Leave Gracefully

On disconnect, the user is marked offline. If the room is empty, it is removed automatically.
